# 📱 AppExistence - Calculatrice d'Existence

Une magnifique application mobile pour calculer la durée d'existence de n'importe quoi!

## 🎯 Fonctionnalités

✨ **Interface élégante et intuitive**
- Design moderne avec dégradés et ombres
- Navigation fluide et responsive
- Emojis pour une meilleure expérience utilisateur

📝 **Fonctionnalités principales**
- Entrez le nom de la chose (ex: "Notre amitié", "La Terre", etc.)
- Sélectionnez la date de début d'existence
- Sélectionnez la date de fin d'existence
- Calculez automatiquement la durée complète

📊 **Résultats détaillés**
L'application affiche:
- Années
- Mois
- Semaines
- Jours
- Heures
- Minutes
- Secondes

Format d'affichage:
```
Votre [chose] a fait [X ans, Y mois, Z semaines, ...] d'existence.
```

## 🚀 Installation et Démarrage

### Prérequis
- Node.js et npm installés
- Expo CLI (optionnel mais recommandé)

### Installation
```bash
cd "c:\Users\JMM\Documents\AppmobilD'existence"
npm install
```

### Démarrage
```bash
npm start
```

Puis, choisissez une des options:
- **`a`** pour Android
- **`i`** pour iOS
- **`w`** pour Web

## 📱 Tester avec Expo Go

1. Téléchargez l'app "Expo Go" sur votre téléphone
2. Scannez le code QR affiché dans le terminal après `npm start`
3. L'application se chargera sur votre téléphone
